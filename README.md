# Money-Manager
Created in Eclipse IDE this is a little money manger that can help keep track of what you do with your money and finances. As you can define different categories, their descriptions and the amount of money spent in that category

*Work In Progess* to Run this manager, Run it in any of your favorite ide's. You will be simply prompted 5 questions answer any of the 1 questions. Have fun managing your money and expenses.
